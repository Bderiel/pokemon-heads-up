# Pokemon Heads Up

Mobile-first Pokemon Heads Up web game.

## Live URL

[https://bderiel.github.io/pokemon-heads-up/](https://bderiel.github.io/pokemon-heads-up/)

## Run Locally

Requirements:
- Node.js 18+
- npm

Steps:
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start local dev server:
   ```bash
   npm run dev
   ```
3. Open the printed local URL in your browser (and on phone via same network if needed).

## Build

```bash
npm run build
```
